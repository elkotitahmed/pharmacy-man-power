# نظام إدارة الحضور وحساب الراتب للصيادلة

تطبيق بسيط باستخدام Streamlit لإدارة حضور الموظفين وحساب الراتب الشهري بناءً على الساعات الفعلية والإجازات المدفوعة.

## المميزات
- تسجيل دخول للمدير والموظفين.
- المدير يستطيع إضافة/تعديل/حذف الموظفين.
- كل موظف يرى جدول حضوره فقط.
- حساب الراتب تلقائياً بناءً على الساعات الفعلية وأيام الإجازة المدفوعة.
- إمكانية تصدير البيانات إلى CSV أو Excel.
- تلوين الأيام: أصفر (إجازة مدفوعة)، أحمر (غياب بدون عذر).

## المتطلبات
- Python 3.8+
- المكتبات المذكورة في `requirements.txt`
